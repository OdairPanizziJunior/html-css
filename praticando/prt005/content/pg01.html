PHP

PHP (acrônimo recursivo para "PHP: Hypertext Preprocessor", originalmente um acrônimo para "Personal Home Page") é uma linguagem de script interpretada, originalmente concebida para programação de páginas web dinâmicas. O interpretador PHP é um software livre distribuído sob a Licença PHP.

Origens

Nascido em 1994 pelo dinamarquês Rasmus Lerdorf, o PHP era originalmente uma coleção de scripts CGI que permitia o gerenciamento fácil de páginas pessoais. De acordo com o anúncio original do PHP 1.0 pelo autor o significado original da sigla era Personal Home Page.

O pacote original foi posteriormente estendido e reescrito pelo próprio Lerdorf em C, adicionando recursos como suporte para o banco de dados mSQL, e passou a ser chamado de PHP/FI, onde FI significa Form Interpreter (interpretador de formulário), proporcionando a possibilidade de integrar código PHP em código HTML a fim de simplificar a criação de páginas dinâmicas. Em 1997, estimou-se que cerca de 50.000 sites eram baseados em PHP.
