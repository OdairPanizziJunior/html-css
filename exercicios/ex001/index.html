Esse é meu primeiro Documento HTML do cusrso em Vídeo.

A sigla HTML é usada para representar o acrônimo para HyperText Markup Language. Lembrando que ela não é uma linguagem de programação e sim uma linguagem de marcação.

Este é um momento em que estou criando o site para demonstrar coisas simples, como por exemplo o uso das Tags H1 para criar os títulos e também da Tag P para a criação dos parágrafos.

Inclusive para deixarmos em negrito, foi utilizada a Tag B que significa "Bold".
